  • Demos

    general:
    Autodesk Viewer
    https://viewer.autodesk.com/ 
    Free viewer site. You can give it a try with your favorite model.

    Viewer Rocks
    https://viewer-rocks.autodesk.io/ 
    Custom viewer site. headless viewer with custom UI and controls to drive the viewer. Gallery of pretranslated models.

    Product Design/Asset Review/M&E
    http://giro-watch.tk/ 
    Using THREE js diectly able to provide transforms / specialized kinematics to test joints etc. Requires extra programming, but for a "family" of assets could be possible to setup previews. Clock was authored in Autodesk Fusion 360.


    AR/VR:
    Forge Viewer option:
    https://vrok.it/v2 
    Uses older WebVR standard. Functionality provided as Viewer extension, so is able to be loaded and enabled at-will.
    AR/VR toolkit option to Unity
    http://forgetoolkit.com/#/ 
    This is a tookit that post-processes the Forge Viewer format into data that can be pulled into Unity. Works in two parts... After using Model Derivative to translated design to Forge Viewer format (SVF), you can then send it to AR VR toolkit services to generate better suited model for Unity and AV VR applications (but not limited). Via a Unity plugin, you can then statically import the asset, or dynamically load it from the server at runtime. This gives advantage of always having latest model in the AR VR experience. Limitations are device based.


    IoT:
    https://forge-rcdb.autodesk.io/configurator?id=58adee163e6f342cf1e92dae 
    Click on a hotspot in the model to show 'streaming IoT data'. Or click on an item in the list on the right. The data is fake - its just a demo. There is a way to connect your laptop to a real sensor and stream the data live to the sample, creating a much more impressive user experience. You need a texas sensor tag and running that project locally, plus a bluetooth connection: https://github.com/leefsmp/sensor-broker


    AEC:
    https://forge-rcdb.autodesk.io/configurator?id=57f3739777c879f48ad54a44 
    Visual Reports
    Multiple Viewers on same page. 3D and 2D. Can change 2D view. Linked geometry data between viewers. Meta data used to build reports using 3rd party JS graph library. Data linked back to geomtry (ie. select a graphing element and show how selection is made in viewer).


    MFG:
    https://forge-rcdb.autodesk.io/database?id=57efaead77c8eb0a560ef465 
    Database/Metadata
    The material list on the top right is a display from an online database. Click a material to see every object made from that material. Open the Properties for an element in the model, scroll down to see the 'Database' properties. These have been injected into the Properties window from the database. Edit the material price in the database and the price in the property window changes too.
    Careful with extensions.... I had some trouble with them recently...
